CSS Fonts

คุณสมบัติ CSS font กำหนดตระกูลแบบอักษรความหนาขนาดและรูปแบบของข้อความ

Font Style

คุณสมบัติ font-style จะใช้เพื่อระบุข้อความ

Example

p.normal {{p.k}}
  font-style: normal;
{{p.l}}

p.italic {{p.k}}
  font-style: italic;
{{p.l}}

p.oblique {{p.k}}
  font-style: oblique;
{{p.l}}

Set Font Size With Pixels

การตั้งค่าขนาดข้อความด้วยพิกเซลช่วยให้คุณควบคุมขนาดข้อความได้อย่างสมบูรณ์:

Example

h1 {{p.k}}
  font-size: 40px;
{{p.l}}

h2 {{p.k}}
  font-size: 30px;
{{p.l}}

p {{p.k}}
  font-size: 14px;
{{p.l}}
