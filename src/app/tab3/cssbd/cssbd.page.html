Border Style

คุณสมบัติ border-styleระบุชนิดของเส้นขอบที่จะแสดง

อนุญาตให้ใช้ค่าต่อไปนี้:

  • dotted - กำหนดเส้นประ
  • dashed - กำหนดขอบประ
  • solid - กำหนดเส้นขอบทึบ
  • double - กำหนดเส้นขอบคู่
  • groove - กำหนดเส้นขอบร่อง 3D ผลกระทบขึ้นอยู่กับค่าสีของเส้นขอบ
  • ridge - กำหนดเส้นขอบเป็นแนว 3 มิติ ผลกระทบขึ้นอยู่กับค่าสีของเส้นขอบ
  • inset - กำหนดเส้นขอบแทรก 3D ผลกระทบขึ้นอยู่กับค่าสีของเส้นขอบ
  • outset - กำหนดเส้นขอบ outset 3D ผลกระทบขึ้นอยู่กับค่าสีของเส้นขอบ
  • none - กำหนดไม่มีขอบ
  • hidden - กำหนดเส้นขอบที่ซ่อนอยู่

คุณสมบัติ border-style สามารถมีได้ตั้งแต่หนึ่งถึงสี่ค่า (สำหรับขอบด้านบน, ขอบด้านขวา, ขอบด้านล่างและขอบด้านซ้าย)

Border Width

คุณสมบัติ border-width ระบุความกว้างของเส้นขอบทั้งสี่

ความกว้างสามารถตั้งค่าเป็นขนาดเฉพาะ (เป็น px, pt, cm, em, ฯลฯ ) หรือโดยใช้หนึ่งในสามค่าที่กำหนดไว้ล่วงหน้า: thin, Medium หรือ thick

คุณสมบัติ border-widthสามารถมีได้ตั้งแต่หนึ่งถึงสี่ค่า (สำหรับขอบด้านบน, ขอบด้านขวา, ขอบด้านล่างและขอบด้านซ้าย)

5px border-width

Example

p.one {{p.k}}
  border-style: solid;
  border-width: 5px;
{{p.l}}

p.two {{p.k}}
  border-style: solid;
  border-width: medium;
{{p.l}}

p.three {{p.k}}
  border-style: solid;
  border-width: 2px 10px 4px 20px;
{{p.l}}

Border Color

คุณสมบัติ border-color ใช้เพื่อตั้งค่าสีของเส้นขอบทั้ง 4

สีสามารถตั้งค่าโดย:

  • name - specify a color name, like "red"
  • Hex - specify a hex value, like "#ff0000"
  • RGB - specify a RGB value, like "rgb(255,0,0)"
  • transparent

คุณสมบัติ border-color สามารถมีได้ตั้งแต่หนึ่งถึงสี่ค่า (สำหรับขอบด้านบน, ขอบด้านขวา, ขอบด้านล่างและขอบด้านซ้าย).
