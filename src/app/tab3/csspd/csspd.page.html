CSS Padding

คุณสมบัติ CSS padding ใช้เพื่อสร้างพื้นที่รอบ ๆ เนื้อหาขององค์ประกอบภายในขอบเขตที่กำหนด

ด้วย CSS คุณสามารถควบคุมการขยายได้อย่างสมบูรณ์ มีคุณสมบัติสำหรับการตั้งค่าการเติมสำหรับแต่ละด้านขององค์ประกอบ (ด้านบนขวาล่างและซ้าย)

Padding - Individual Sides

CSS มีคุณสมบัติในการระบุการเติมสำหรับแต่ละด้านขององค์ประกอบ

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

คุณสมบัติ padding ทั้งหมดสามารถมีค่าต่อไปนี้:

  • length - ระบุการเติมเต็มใน px, pt, cm, ฯลฯ
  • % - ระบุการเติม % ของความกว้างขององค์ประกอบที่มี

Example

div {{p.k}}
  padding-top: 50px;
  padding-right: 30px;
  padding-bottom: 50px;
  padding-left: 80px;
{{p.l}}

หากคุณสมบัต padding มี 4 ค่า:

  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px
    • left padding is 100px

Example

div {{p.k}}
  padding: 25px 50px 75px 100px;
{{p.l}}
  • padding: 25px 50px 75px;
    • top padding is 25px
    • right and left paddings are 50px
    • bottom padding is 75px

Example

div {{p.k}}
  padding: 25px 50px 75px;
{{p.l}}
  • padding: 25px 50px;
    • top and bottom paddings are 25px
    • right and left paddings are 50px

Example

div {{p.k}}
  padding: 25px 50px;
{{p.l}}
  • padding: 25px;
    • all four paddings are 25px

div {{p.k}}
