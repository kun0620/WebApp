เมื่อเบราว์เซอร์อ่าน style sheet มันจะจัดรูปแบบเอกสาร HTML ตามข้อมูลใน style sheet.

3 วิธีในการแทรก CSS

การแทรก style sheet มี 3 วิธี

  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet

ด้วย style sheet ภายนอกคุณสามารถเปลี่ยนรูปลักษณ์ของทั้งเว็บไซต์ได้ด้วยการเปลี่ยนไฟล์เพียงไฟล์เดียว!

แต่ละหน้าจะต้องมีการอ้างอิงไปยังไฟล์ style sheet ภายนอกภายในองค์ประกอบ <link> องค์ประกอบ <link> อยู่ในส่วน <head>

Example

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

style sheet ภายนอกสามารถเขียนได้ในโปรแกรมแก้ไขข้อความใด ๆ ไฟล์ไม่ควรมีแท็ก html ใด ๆ ไฟล์ style sheet ต้องถูกบันทึกด้วยนามสกุล .css

นี่คือลักษณะ "mystyle.css"

body
  background-color: lightblue;


h1 {
  color: navy;
  margin-left: 20px;
}

หมายเหตุ: อย่าเพิ่มช่องว่างระหว่างค่าคุณสมบัติและหน่วย (margin-left: 20 px;) วิธีที่ถูกต้องคือ: margin-left: 20px;
