CSS How To...

เมื่อเบราว์เซอร์อ่าน style sheet มันจะจัดรูปแบบเอกสาร HTML ตามข้อมูลใน style sheet.

3 วิธีในการแทรก CSS

การแทรก style sheet มี 3 วิธี

  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet

ด้วย style sheet ภายนอกคุณสามารถเปลี่ยนรูปลักษณ์ของทั้งเว็บไซต์ได้ด้วยการเปลี่ยนไฟล์เพียงไฟล์เดียว!

แต่ละหน้าจะต้องมีการอ้างอิงไปยังไฟล์ style sheet ภายนอกภายในองค์ประกอบ <link> องค์ประกอบ <link> อยู่ในส่วน <head>

Example

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

style sheet ภายนอกสามารถเขียนได้ในโปรแกรมแก้ไขข้อความใด ๆ ไฟล์ไม่ควรมีแท็ก html ใด ๆ ไฟล์ style sheet ต้องถูกบันทึกด้วยนามสกุล .css

นี่คือลักษณะ "mystyle.css"

body {{p.k}}
  background-color: lightblue;
{{p.l}}

h1 {{p.k}}
  color: navy;
  margin-left: 20px;
{{p.l}}

หมายเหตุ: อย่าเพิ่มช่องว่างระหว่างค่าคุณสมบัติและหน่วย (margin-left: 20 px;) วิธีที่ถูกต้องคือ: margin-left: 20px;

Internal Style Sheet

อาจใช้ style sheet ภายในหากหน้าเดียวมี styleที ่ไม่ซ้ำกัน

style ภายในถูกกำหนดไว้ภายในองค์ประกอบ <style> ภายในส่วน <head> ของหน้า HTML:

Example

<head>
<style>
body {{p.k}}
  background-color: linen;
{{p.l}}

h1 {{p.k}}
  color: maroon;
  margin-left: 40px;
{{p.l}}
</style>
</head>

Inline Styles

อาจใช้ inline style เพื่อใช้สไตล์ที่เป็นเอกลักษณ์สำหรับองค์ประกอบเดียว

ในการใช้รูปแบบ inline ให้เพิ่มแอททริบิวสไตล์เพื่อองค์ประกอบที่เกี่ยวข้อง แอตทริบิวต์ลักษณะสามารถมีคุณสมบัติ CSS ใดๆ

ตัวอย่างด้านล่างแสดงวิธีการเปลี่ยนสีและขอบซ้ายขององค์ประกอบ <h1>

Example

<h1 style="color:blue;margin-left:30px;">This is a heading</h1>

style sheet หลายแผ่น หากมีการกำหนดคุณสมบัติบางอย่างสำหรับตัวเลือกเดียวกัน (องค์ประกอบ) ใน style sheet ที่ต่างกันจะใช้ค่าจาก style sheet การอ่านล่าสุด
