CSS Backgrounds

คุณสมบัติ CSS background ใช้เพื่อกำหนดเอฟเฟกต์พื้นหลังสำหรับองค์ประกอบ คุณสมบัติ CSS background :

CSS background properties:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Background Color

คุณสมบัติ background-color ระบุสีพื้นหลังขององค์ประกอบ

สีพื้นหลังของหน้าถูกตั้งค่าดังนี้:

Example

body {{p.k}}
  background-color: lightblue;
{{p.l}}

Background Image

คุณสมบัติ background-image ระบุภาพที่จะใช้เป็นพื้นหลังขององค์ประกอบ

โดยค่าเริ่มต้นรูปภาพจะถูกทำซ้ำเพื่อให้ครอบคลุมองค์ประกอบทั้งหมด

ภาพพื้นหลังสำหรับหน้าสามารถตั้งค่าดังนี้:

Example

body {{p.k}}
  background-image: url("paper.gif");
{{p.l}}

Background Image - Repeat

หากภาพถูกทำซ้ำในแนวนอน (background-repeat: repeat-x;), พื้นหลังจะดูดีขึ้น:

Example

body {{p.k}}
  background-image: url("gradient_bg.png");
  background-repeat: repeat-x;
{{p.l}}

Tip: ในการทำซ้ำภาพในแนวตั้งให้ตั้งค่าพื้นหลังซ้ำ: background-repeat: repeat-y;

Background Image - Set position and no-repeat

รูปภาพพื้นหลังจะแสดงในตำแหน่งเดียวกับข้อความ เราต้องการเปลี่ยนตำแหน่งของภาพเพื่อไม่ให้รบกวนข้อความมากเกินไป

ตำแหน่งของภาพถูกระบุโดยคุณสมบัติ background-position

Example

body {{p.k}}
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
{{p.l}}

Background Image - Fixed position

ในการระบุว่า background image ควรได้รับการแก้ไข (จะไม่เลื่อนไปกับส่วนที่เหลือของหน้า) ให้ใช้คุณสมบัติการแนบภาพพื้นหลัง: background-attachment

Example

body {{p.k}}
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: fixed;
{{p.l}}
