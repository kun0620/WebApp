Styling Links

Links สามารถกำหนดสไตล์ด้วยคุณสมบัติ CSS

Example

a {{p.k}}
  color: hotpink;
{{p.l}}

นอกจากนี้ลิงก์ยังสามารถจัดสไตล์ได้แตกต่างกันไปตามสถานะของลิงค์

สถานะการเชื่อมโยงสี่สถานะคือ:

  • a:link - ลิงค์ปกติที่ไม่เคยเข้าชม
  • a:visited - ลิงค์ที่ผู้ใช้เข้าชม
  • a:hover - ลิงก์เมื่อผู้ใช้เลื่อนเมาส์ไว้เหนือมัน
  • a:active - ลิงค์ทันทีที่มีการคลิก

Example

/* ลิงค์ปกติที่ไม่เคยเข้าชม */
a:link {{p.k}}
  color: red;
{{p.l}}

/* ลิงค์ที่ผู้ใช้เข้าชม */
a:visited {{p.k}}
  color: green;
{{p.l}}

/* ลิงก์เมื่อผู้ใช้เลื่อนเมาส์ไว้เหนือมัน */
a:hover {{p.k}}
  color: hotpink;
{{p.l}}

/* ลิงค์ทันทีที่มีการคลิก */
a:active {{p.k}}
  color: blue;
{{p.l}}

Advanced - Link Buttons

ตัวอย่างนี้แสดงตัวอย่างขั้นสูงเพิ่มเติมที่เรารวมคุณสมบัติ CSS หลายอย่างเพื่อแสดงลิงก์เป็นกล่อง / ปุ่ม:

Example

a:link, a:visited {{p.k}}
  background-color: #f44336;
  color: white;
  padding: 14px 25px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
{{p.l}}

a:hover, a:active {{p.k}}
  background-color: red;
{{p.l}}
