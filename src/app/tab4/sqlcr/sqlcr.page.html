AngularJS Controllers

แอปพลิเคชัน AngularJS ควบคุมโดย controllers

คำสั่ง ng-controller กำหนดแอพพลิเคชั่น controller.

controller เป็นวัตถุ JavaScript Objectโดยตัวสร้างวัตถุ standard JavaScript object constructor..

AngularJS Example

<div ng-app="myApp" ng-controller="myCtrl">

First Name: <input type="text" ng-model="firstName"><br>
Last Name: <input type="text" ng-model="lastName"><br>
<br>
Full Name: {{firstName + " " + lastName}}

</div>
