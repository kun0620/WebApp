HTML Links

Link พบได้ทุกหน้าเว็บ Link อนุญาตให้ผู้ใช้คลิกทางจากหน้าหนึ่งไปอีกหน้าหนึ่ง

HTML Links - Hyperlinks

คุณสามารถคลิกที่ลิงค์และข้ามไปที่หน้าอื่น

เมื่อคุณเลื่อนเมาส์ไปที่ลิงค์ลูกศรของเมาส์จะกลายเป็นรูปอื่นได้

หมายเหตุ: ลิงค์ไม่จำเป็นต้องเป็นข้อความ มันอาจเป็นรูปภาพหรือองค์ประกอบ HTML อื่น ๆ

ใน HTML ลิงก์จะถูกกำหนดด้วยแท็ก <a>tag:

<a href="url">link text</a>

Example

<a href="https://ionicframework.com">Visit our HTML tutorial</a>

แอตทริบิวต์ href ระบุที่อยู่ปลายทาง (https://ionicframework.com) ของลิงก์

ข้อความลิงค์เป็นส่วนที่มองเห็นได้ (Visit our HTML tutorial)

คลิกที่ลิงค์ข้อความจะส่งคุณไปยังที่อยู่ที่ระบุ

Local Links

ตัวอย่างข้างต้นใช้ URL แบบสมบูรณ์ (ที่อยู่เว็บแบบเต็ม)

มีการระบุลิงก์ในตัวเครื่อง (ลิงก์ไปยังเว็บไซต์เดียวกัน) ด้วย URL แบบสัมพัทธ์ (ไม่มี https: // www .... )

Example

<a href="html_Link.html">HTML Images</a>

HTML Links - Image as Link

Example

<a href="default.asp">
  <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;border:0;">
</a>

HTML Links - Create a Bookmark

ใช้เพื่ออนุญาตให้ผู้อ่านข้ามไปยังบางส่วนของเว็บเพจ บุ๊กมาร์กอาจมีประโยชน์หากหน้าเว็บของคุณยาวมาก ในการสร้างบุ๊คมาร์คคุณต้องสร้างบุ๊กมาร์กก่อนจากนั้นจึงเพิ่มลิงค์ลงไป เมื่อคลิกที่ลิงค์หน้าจะเลื่อนไปยังตำแหน่งที่มีบุ๊คมาร์ค

Example

ก่อนอื่นให้สร้างบุ๊คมาร์คด้วย id attribute:

<h2 id="top">Top</h2>

จากนั้นเพิ่มลิงก์ไปยังบุ๊คมาร์ค ("Jump to Top") จากภายในหน้าเดียวกัน:

<a href="#C4">Jump to Top</a>
