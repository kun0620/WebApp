HTML Documents

HTML ทั้งหมดจะต้องเริ่มต้นด้วยการประกาศ {{p.d}}

HTML นั้นเริ่มต้นด้วย {{p.hk}} และลงท้ายด้วย {{p.h}}

ส่วนที่มองเห็นได้ของ HTML อยู่ระหว่าง {{p.bk}} และ {{p.b}}

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
