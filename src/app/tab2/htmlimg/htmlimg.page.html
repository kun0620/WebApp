HTML Images

รูปภาพสามารถปรับปรุงการออกแบบและลักษณะที่ปรากฏของเว็บเพจ

ใน HTML รูปภาพจะถูกกำหนดด้วยแท็ก <img> tag.

แท็ก <img> นั้นว่างเปล่ามันมีแอตทริบิวต์เท่านั้นและไม่มีแท็กปิด

แอตทริบิวต์ src ระบุ URL (ที่อยู่เว็บ) ของภาพ:

<img src="url">

Image Size - Width and Height

คุณสามารถใช้ style เพื่อระบุความกว้างและความสูงของรูปภาพ

Example

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

หรือสามารถใช้ width และ height attributes:

Example

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Images in Another Folder

ตามเป็นเรื่องปกติที่จะเก็บรูปภาพไว้ในโฟลเดอร์ย่อย จากนั้นคุณจะต้องรวมชื่อโฟลเดอร์ในแอตทริบิวต์src attribute:

Example

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Images on Another Server

บางเว็บไซต์เก็บภาพของพวกเขาบนเซิร์ฟเวอร์ภาพ

จริงๆแล้วคุณสามารถเข้าถึงภาพจากที่อยู่เว็บใด ๆ ในโลก:

Example

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">
