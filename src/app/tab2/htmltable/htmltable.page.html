การกำหนดตาราง HTML

ตาราง HTML ถูกกำหนดด้วยแท็ก<table>

แต่ละแถวของตารางจะถูกกำหนดด้วยแท็ก <tr> ส่วนหัวของตารางถูกกำหนดด้วยแท็ก <th> tag. โดยค่าเริ่มต้นส่วนหัวของตารางจะหนาและอยู่ตรงกลาง ข้อมูลตาราง / เซลล์ถูกกำหนดด้วยแท็ก <td>

Example

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>
